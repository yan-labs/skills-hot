# @skills-hot/shot

The official CLI for [Skills Hot](https://skills.hot) - Discover, install, and share AI agent skills.

## Installation

```bash
# Using npx (no installation required)
npx @skills-hot/shot --help

# Global installation
npm install -g @skills-hot/shot
```

## Usage

```bash
# Search for skills
shot search git

# Install a skill
shot add git-commit

# Login to your account
shot login

# Publish a skill
shot publish

# Get help
shot --help
```

## Quick Install (Alternative)

If you prefer to install the native binary directly:

```bash
# macOS/Linux
curl -fsSL https://skills.hot/install.sh | bash

# Windows PowerShell
irm https://skills.hot/install.ps1 | iex
```
